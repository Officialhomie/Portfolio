# Project Images

Place your project images in this directory. Each project should have a corresponding image file.

## Required Images:

1. **multisig-wallet.png** - MultiSig Wallet project
2. **dx-bloom.png** - DX Bloom project
3. **oneseed.png** - OneSeed project
4. **healthtrove.png** - HealthTrove project
5. **ifindr.png** - iFindr project
6. **education-chain.png** - EducationChain project
7. **dex-dapp.png** - DEX dApp Template project
8. **cavefi.png** - CaveFi project
9. **bridge-scope.png** - Bridge Scope project
10. **paysteam.png** - PaySteam project

## Image Specifications:

- **Format**: PNG or JPG
- **Recommended Size**: 1200x675px (16:9 aspect ratio)
- **File Size**: Keep under 500KB for optimal loading
- **Naming**: Use lowercase with hyphens (e.g., `multisig-wallet.png`)

## Featured Projects (Priority):

These 3 projects are featured on the homepage and should have high-quality images:
1. multisig-wallet.png
2. dx-bloom.png
3. oneseed.png

## Fallback:

If an image is missing, the component will show a placeholder icon. However, it's recommended to add actual project screenshots or logos for better presentation.
